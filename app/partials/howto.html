Welcome to forms-angular

This demo application uses the following models:

  • {{model.resource_name}} [New][List]

For each model the following routes are supported:

  1. /#/:model which lists the documents in the collection, with links to the form for editing them
  2. /#/:model/new which enables the user to create a new document using the default form for the collection
  3. /#/:model/:id/edit which enables the user to edit a document

The Angular controllers make use of the following API calls:

  1. /api/models returns an array of all the models that are available to forms-angular
  2. /api/schema/:model returns the schema for a model
  3. /api/:model returns an array of all the documents in a model (needs pagination)
  4. /api/:model/:id returns a single document from the collection
